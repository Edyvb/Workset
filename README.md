## 🚀 Gestionnaire d'Espaces de Travail  

Application Electron (HTML/CSS/JS) permettant de créer, gérer et lancer des espaces de travail personnalisés pour automatiser l'ouverture d'outils et de services nécessaires à un projet.

**Statut** : _En cours de développement_

---

## 📌 Fonctionnalités  

- 📂 **Gestion des Espaces de Travail** : Crée, modifie et supprime des espaces.  
- 🔄 **Ouverture Automatisée** : Lance automatiquement VS Code, un terminal, un serveur, etc.  
- ⚙️ **Tâches Personnalisées** : Exécute des commandes spécifiques (executer une commande, lancer une base de données...).  
- 🖥️ **Objectif : création d'un exécutable compatible Windows & Linux**.  

---

## ⚡ Exemple de Tâche Personnalisée

- **Démarrer un serveur local**  
  ```bash
  python -m http.server 8000
  ```

## 🎯 Objectifs à venir  
- 🎨 **Interface Graphique** pour une gestion plus intuitive.  
- 🏗 **Création d'un exécutable pour Windows et Linux**.  
- 🔄 **Sauvegarde et synchronisation des espaces entre plusieurs machines**.
